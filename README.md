# Student Record System (CLI)
A command-line based application built using Python to manage student records.  
The system allows users to add, search, update, and delete student data with persistent file storage.

## Features
- Add student records (name and marks)
- View all students
- Search student by name
- Update student marks
- Delete student records
- Clear all records
- Persistent storage using file handling
- Input validation and duplicate prevention

## Tech Used
- Python
- File Handling

## How to Run
1. Download or clone the repository  
2. Open terminal in the project folder  
3. Run:
python main.py

## Example Usage
1. Add a new student with marks  
2. Search or update existing records  
3. Delete or clear records as needed  

All data is stored and loaded from a file automatically.

## Project Highlights
- Built using core Python without external libraries  
- Implements CRUD operations for structured data  
- Handles invalid inputs and edge cases  
- Designed for simplicity and efficiency in CLI environment
